<strong>WP Cookie User Info</strong>  is a <strong>FREE WordPress Plugin</strong> which allows you to apply cookie information on your website on different positions. Its fun because - you can create, customize and build the cookie info for your site on your own, choose on what to display as a message, where to display it and reappear after how long.

<strong>You can select the built in templates and use a custom display setting of your choice, which includes choosing from different colors, font families and sizes.</strong>

== Installation ==
1. Unzip wp-cookie-user-info.zip
1. Upload all the files to the /wp-content/plugins/wp-cookie-user-info
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. For customizing the plugin's settings, click on Add New Settings, Selected Settings or Activated Settings in Wordpress left admin menu.


== Frequently Asked Questions ==
= What does this plugin do? =
This plugin provides the ability to create, customize and build beautiful cookie info notices for your site on your own.

= Can I choose how the cookie bar looks ? =
Our settings is composed of Custom Template Settings where you can make a custom template that will store the color values, fonts and other proprties for the different components in your cookie notice, and use it in cookie notice that uses these custom templates.

= Do I have to make a Cookie Notice to start using the Plugin ? =
Yes, creating a Cookie Notice is easy, and requires minimal knowledge. You are also familiarized with our plugins interface this way.

= What isn't this plugin ? = 
This plugin isn't like regular notification bar plugins, where you can add almost anything your heart desires. Cookie Notice specifically informs the site visitors about the use of Cookies in your site. Furthermore, it doesn't allow blocking specific or all cookies for the visitors.
